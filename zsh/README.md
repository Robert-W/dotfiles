# ZSH Setup
zsh configurations as of 12/07/22

## Core requirements
- iTerm2

## Brew packages
- `brew install iterm2`

## Clone and Link configuration files
- `git clone git@github.com:Robert-W/dotfiles.git`
- `ln -s "$(pwd)/zsh/.zshrc" ~/.zshrc`

> May need to delete the `.zshrc` file if one already exists.
