# ZSH Setup
zsh configurations as of 12/07/22

## Core requirements
- iTerm2

## Brew packages
- `brew install iterm2`

## Link configuration files
- `ln -s "$(pwd)/zsh/.zfunc" ~/.zfunc`
- `ln -s "$(pwd)/zsh/.zshrc" ~/.zshrc`
- `ln -s "$(pwd)/zsh/.zprofile" ~/.zprofile`

> May need to delete the `.zshrc` file if one already exists.
